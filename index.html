All Commands of git/Github

Git - Git is basically a VCS(Version Control System). It tracks the history of changes as people and team collaborated on projects together. As the project evolves, team can run, test, fix bugs and contribute new code with confidence that any version can be recovered at any time. Developers can review History to find out -

  1. Which Changes were made
  2. Who made changes?
  3. When the changes made?
  4. Why were changes needed?
